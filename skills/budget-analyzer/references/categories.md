# Budget Categories

## Standard Categories

### Essential
- **Housing** - Rent, mortgage, property taxes
- **Utilities** - Electric, water, gas, internet
- **Groceries** - Food, household supplies
- **Transportation** - Fuel, public transport, parking

### Lifestyle
- **Dining** - Restaurants, takeout, coffee
- **Entertainment** - Movies, concerts, streaming
- **Shopping** - Clothing, electronics, misc
- **Health** - Medical, pharmacy, fitness

### Financial
- **Savings** - Emergency fund, investments
- **Debt** - Credit cards, loans
- **Insurance** - Health, car, home, life

### Subscriptions
- **Streaming** - Netflix, Spotify, etc.
- **Software** - Adobe, Microsoft, etc.
- **Memberships** - Gym, clubs, etc.

## Custom Categories

You can add custom categories via:

```javascript
budgetAnalyzer.addCategory({
  name: 'Pet Care',
  budget: 100,
  recurring: true
});
```
